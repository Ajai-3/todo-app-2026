## ⏲ Core Features

###  Advanced Focus System
- **Unified NavTimer**: A persistent Stopwatch/Timer hybrid built directly into the navigation bar.
- **Custom Duration**: Set focus sessions with 1m/5m/10m presets or custom user-defined minutes.
- **Task Timer Logging**: Log millisecond-precise time directly on individual task cards.
- **Audio Haptics**: Professional alert sounds for focus start, edit feedback, and task completion.

###  Deep Analytics & Tracking
- **Streak Calendar**: A heatmap-style productivity map with pulsing fire icons to track your daily consistency.
- **Category Performance**: Horizontal gradient visualizations for task distribution.
- **Productivity Spider-Graph**: High-end data visualization to ensure your focus is balanced across life categories.
- **Priority Donuts**: Visual breakdown of your most urgent commitments.

###  Professional Task Management
- **Kanban Board**: Drag-and-drop task status updates with smooth animations.
- **Local-First Design**: Your data stays in your browser. Fast, offline-capable, and private.
- **Recurring Tasks**: Sophisticated system for daily or weekly habit templates.
- **Smart Warnings**: Integrated audio alerts if a task timer exceeds its estimated budget.

###  Design Aesthetic
- **Deep Black UI**: Minimalist, high-contrast dark theme with glowing Emerald Green accents.
- **Custom Scrollbars**: Emerald Green themed scrollbars for a cohesive experience.
- **Premium Animations**: Shimmering flares on stats cards and smooth Framer Motion transitions.

##  Local Installation

Get the Todo dashboard running on your machine in under 2 minutes:

1. **Clone the repository:**
   ```bash
   git clone [repository-url]
   cd todo-app-2026
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Launch the development server:**
   ```bash
   npm run dev
   ```

4. **Access the Dashboard:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.

---

##  Usage Tips
- **Warning Beeps**: If you see (and hear) the timer beeping for 5 seconds on a task, it means you've exceeded your estimated time! ⚠️
- **Sound Toggle**: Use the volume icon in the navbar to quickly mute/unmute focusing alerts.
- **Drag & Drop**: In the **Board View**, simply drag tasks between "Pending", "In Progress", and "Completed" to update your workflow.
